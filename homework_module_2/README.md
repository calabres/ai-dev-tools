# Online Coding Interview Platform

A real-time collaborative coding platform for technical interviews.

## Prerequisites

- Node.js (v18+)
- npm

## Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd homework_module_2
    ```

2.  **Install Backend Dependencies:**
    ```bash
    cd backend
    npm install
    ```

3.  **Install Frontend Dependencies:**
    ```bash
    cd ../frontend
    npm install
    ```

## Running the Application

### Backend
Start the backend server (runs on port 3000):
```bash
cd backend
node server.js
```

### Frontend
Start the frontend development server (runs on port 5173):
```bash
cd frontend
npm run dev
```

## Running Tests

Run integration tests for the backend:
```bash
cd backend
npm test
```
